# Contributing

Grazie per il contributo!

## Flusso rapido
- Crea un branch dal main.
- Descrivi chiaramente lo scopo della PR.
- Aggiorna la documentazione se tocchi dipendenze o setup.

## Dipendenze
- Segui le regole in `DEPENDENCIES.md`.
- Riporta in PR la motivazione della nuova libreria.

## Convenzioni
- Mantieni le cartelle esistenti: `backend/` e `frontend/`.
- Evita cambi strutturali grandi senza una issue approvata.
